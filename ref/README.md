# `ref/` — 参考与归档

本目录**不参与**主工程运行；用于保留旧版实现、脱敏数据样例或本地备份，便于对照与迁移。

| 路径 | 说明 |
|------|------|
| `toolboxweb/` | 旧版 Flask 工具箱源码（重构时的行为与页面参考；见 `docs/MOS_TOOLBOX_REBUILD_BASELINE.md` 的收敛范围）。 |
| `archive/` | 人工归档：例如本地 SQLite 备份等；见该子目录内说明。 |

不要将生产密钥、明文账号密码提交到版本库。`toolboxweb/static/config/` 等路径下可能存在历史账号/密码类 JSON，**勿提交真实内容**；若需保留目录结构，可改为示例文件或仅本地保留。
